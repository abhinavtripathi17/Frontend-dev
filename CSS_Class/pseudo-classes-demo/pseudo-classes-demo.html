CSS Pseudo-Class Demonstration

This page demonstrates the use of :hover, :focus, :active, and :nth-child() pseudo-classes using links, buttons, and lists.

:hover

Hover over me

The :hover pseudo-class applies when you move your mouse over an element.

:focus

The :focus pseudo-class is activated when an input field is clicked or tabbed into.

:active

The :active pseudo-class works when you press and hold an element like a button or link.
